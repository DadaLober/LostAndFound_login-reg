# Simple login and register function authentication with database

## Installation:

**Prerequisites:**
- PHP installed
- XAMPP (to test the code locally)

**Steps:**

- Create a database named **appdb**
- Import **tblusers.sql** to your database **(appdb)**
- Host locally in XAMPP htdocs folder
